# 公务员考试模拟答题系统

一个基于浏览器本地运行的公务员考试模拟答题系统，无需后端服务器，双击 HTML 文件即可使用。

## 功能特点

### 普通用户功能
- 📝 用户注册与登录
- ✏️ 在线答题（支持 5 个模块）
- 📋 答题历史记录
- 📊 统计分析（按模块统计正确率）

### 管理员功能
- ⚙️ 题库管理（添加、编辑、删除题目）
- 📑 按模块筛选题目

## 模块分类

| 模块 | 说明 |
|------|------|
| 言语理解 | 考察文字理解和表达能力 |
| 数量关系 | 考察数学运算和逻辑思维 |
| 判断推理 | 考察逻辑判断和推理能力 |
| 常识判断 | 考察综合知识储备 |
| 资料分析 | 考察数据分析和计算能力 |

## 快速开始

1. 用浏览器打开 `index.html`
2. 使用默认管理员账号登录，或注册新用户

### 默认管理员账号
- 用户名：`admin`
- 密码：`admin123`

## 技术实现

- **存储方式**：localStorage 本地存储
- **运行环境**：现代浏览器（Chrome、Edge、Firefox 等）
- **依赖**：无第三方依赖，纯原生 JavaScript

## 使用说明

1. **注册账号**：点击注册标签，填写用户名、密码，选择角色（普通用户/管理员）
2. **开始答题**：选择相应模块，进行答题练习
3. **查看统计**：在统计分析页面查看各模块正确率
4. **管理题库**：管理员可在题库管理页面添加、编辑、删除题目

## 注意事项

- 数据存储在浏览器本地，清除浏览器缓存会导致数据丢失
- 不同浏览器之间的数据不互通
- 建议在固定浏览器中使用
